## ğŸ“ Customization

### Add Your Resume

1. Save your resume as `Umer-Khan-Resume.pdf`
2. Place it in the `resume/` folder
3. The download link is already configured in the HTML

### Add Project Screenshots

1. Capture screenshots of your live Shopify stores
2. Save them to the `images/` folder
3. Update image references in `index.html` and `projects.html`

Example:
```html
<!-- Replace gradient placeholders with actual images -->
<img src="images/polawn-screenshot.jpg" alt="Polawn Store" class="project-image">
```

### Update Contact Information

Edit `contact.html`:
- Email address
- LinkedIn URL
- Any other contact details
